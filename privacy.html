Privacy Policy

Le Indagini — Giochi di deduzione storici — Ultimo aggiornamento: marzo 2026

L'applicazione

Le Indagini è un'applicazione web statica (PWA) che non raccoglie, trasmette né elabora dati personali degli utenti. Non sono presenti form di registrazione, account, analytics di terze parti, cookie di profilazione o pubblicità.

Hosting — GitHub Pages

Questa applicazione è ospitata tramite GitHub Pages. Quando visiti il sito, GitHub Inc. raccoglie automaticamente alcune informazioni tecniche, tra cui l'indirizzo IP del visitatore, per motivi di sicurezza e operativi. Questa raccolta è gestita interamente da GitHub e non è accessibile all'autore del gioco.

Per i dettagli su come GitHub tratta questi dati, consulta:
GitHub General Privacy Statement
GitHub Pages — Data Collection

Controllo aggiornamenti automatico

All'avvio dell'applicazione, viene effettuata automaticamente una richiesta al file version.json ospitato su GitHub Pages per verificare la disponibilità di nuovi contenuti. Questa richiesta comporta la trasmissione dell'indirizzo IP del dispositivo ai server di GitHub, secondo le modalità descritte nella loro privacy policy.

Questa operazione avviene automaticamente senza azione da parte dell'utente. Lo scopo è esclusivamente verificare la presenza di aggiornamenti di gioco. Nessun dato viene trasmesso all'autore.

Service Worker e cache

L'applicazione utilizza un service worker per funzionare offline. I file di gioco (pagine HTML, icone) vengono memorizzati nella cache del browser sul dispositivo dell'utente. Questi dati restano locali e non vengono inviati a nessun server.

Archiviazione locale (localStorage)

L'applicazione utilizza il localStorage del browser per memorizzare sul dispositivo dell'utente la versione dell'aggiornamento già visualizzata, in modo da non mostrare lo stesso avviso più volte. Questo dato non lascia mai il dispositivo e non è accessibile a nessun server o terza parte.

Dati dei minori

L'applicazione non raccoglie consapevolmente dati di minori. Trattandosi di contenuti storici a tema investigativo, è consigliata per un pubblico adulto o con supervisione.
